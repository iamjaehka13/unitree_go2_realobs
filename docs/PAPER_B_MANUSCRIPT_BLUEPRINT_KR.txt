Paper B 원고 청사진 (2026-03-09)
===============================

제목
----
후보 1. Real-Observable Reinforcement Learning for Robust Locomotion under Motor Degradation

현재는 위 두 제목 후보 중 하나를 최종 선택하기 전 단계로 둔다.


0) 이 문서의 목적
-----------------
이 문서는 현재 코드 표면과 논문 서사를 하나로 잠그기 위한 active blueprint다.
핵심은 아래 세 가지를 동시에 고정하는 것이다.

1. 무엇을 주장할 것인가
2. 어떤 실험 사다리로 그 주장을 입증할 것인가
3. 무엇은 아직 주장하면 안 되는가


1) 최종 문제정의
----------------
Go2 locomotion policy가 motor degradation, temperature rise, voltage sag가 존재하는 조건에서도
실측 가능한 observable channels만으로 안전-성능 균형을 유지할 수 있는가.

논문 B의 초점은 "더 많은 숨은 상태를 쓰는가"가 아니다.
"실제로 배치 가능한 관측만으로 얼마나 회복되는가"가 핵심이다.


2) 메인 논지
------------
한 줄 claim:

measurable-only real-observable design can recover meaningful safety benefits under motor degradation,
even without privileged degradation state access.

이 문장을 더 구체화하면 아래와 같다.

1. `Baseline` 대비 `RealObs-ObsOnly`가 measurable-only 관측 자체의 가치를 보여준다.
2. `RealObs-ObsOnly` 대비 `RealObs-Full`이 observable safety reward의 추가 기여를 보여준다.
3. `Strategic`은 deployment policy가 아니라 privileged upper bound다.
4. `TempDose`는 메인 claim이 아니라 thermal semantics side ablation이다.


3) 메인 비교 사다리
-------------------
본문의 core quartet은 아래로 고정한다.

1. `Baseline`
   same degraded physics, no hidden degradation observations, no degradation-aware rewards
2. `RealObs-ObsOnly`
   measurable-only observations, baseline rewards
3. `RealObs-Full`
   measurable-only observations, observable safety rewards
4. `Strategic`
   privileged upper bound

사이드 실험:

1. `RealObs-TempDose`
2. `RealObs-Full governor off/on`
3. `RealObs-Full` observation ablations
4. `RealObs-Full` sensor realism ablations

현재 task ID 매핑:

1. `Unitree-Go2-Baseline-v1`
2. `Unitree-Go2-ObsOnly-v1`
3. `Unitree-Go2-RealObs-v1`
4. `Unitree-Go2-Strategic-v1`
5. `Unitree-Go2-TempDose-v1`


4) Contribution 문장 초안
-------------------------
Contribution 1.
We formulate motor-degradation locomotion as a real-observable safety problem rather than a privileged state estimation problem.

Contribution 2.
We provide a comparison ladder that separates the effect of measurable observations from the effect of safety-oriented reward shaping.

Contribution 3.
We couple simulation evaluation with real-log governor analysis so that thermal and voltage margins are discussed with deployment-facing metrics.

10) Main claim과 non-claim
--------------------------
본문에서 가능한 주장:

1. measurable-only observations만으로도 degradation safety benefit을 회복할 수 있다
2. observable safety reward를 추가하면 그 benefit이 더 커진다
3. privileged upper bound와의 gap을 정량적으로 제시할 수 있다
4. real-log 기반 governor analysis로 threshold semantics를 더 deployment-facing하게 해석할 수 있다

본문에서 금지할 주장:

1. 이미 실로봇에서 full deployment robustness가 검증됐다는 식의 강한 일반화
2. `Strategic`가 배치 가능한 정책이라는 서술
3. 보정 전 threshold 절대값의 보편성 주장
4. `TempDose`를 main novelty로 끌어올리는 서술

12) 실로봇 정렬 파트의 역할
---------------------------
실로봇 파트는 아래를 보강하는 용도다.

1. governor on/off safety behavior
2. thermal/voltage metric semantics alignment
3. raw 500Hz log와 50Hz step log 연결
4. threshold calibration을 위한 empirical evidence

실로봇 파트가 아직 하지 않는 일:

1. 논문 main policy superiority를 단독으로 증명하는 것
2. 모든 scenario를 fully reproduce하는 것

15) 제출 전 체크리스트
---------------------
1. 모든 메인 표가 quartet을 기준으로 정렬되어 있는가
2. `ObsOnly`와 `RealObs`의 차이가 reward contribution으로 명확히 읽히는가
3. `Strategic`가 upper bound로만 소개되는가
4. `TempDose`가 메인 표를 흐리지 않는가
5. 실로봇 로그가 threshold semantics 논리를 보강하는가
6. 본문 어디에도 옛 distillation/replay 중심 서사가 남아 있지 않은가
